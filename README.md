Requirements:
  XAMPP

Initialize:
1. Git Clone the repository into xampp/htdocs
2. Run XAMPP, start Apache & MySQL
3. Open Command Prompt, run MYSQL
4. Follow the commands in database.sql
5. Follow the commands in values.sql
6. Go to localhost/proj/index.html on the browser

Unhashed Passwords of Users:

'admin' for User in admin role

'therapist' for Users in therapist role

'customer' for Users in customer role
